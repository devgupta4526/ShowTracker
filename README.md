

# 🎬 ShowTracker | Android, Java, Retrofit, Firebase

ShowTracker is an Android app designed to help users easily track the shows they’ve watched, are currently watching, and plan to watch. It offers a clean, user-friendly interface that makes organizing your movies simple and enjoyable.

---

### 🛠 **Built With:**
- **Java** - Primary language for the application
- **Retrofit** - For seamless API integration with TMDB
- **Firebase Realtime Database** - To store and manage user data
- **Material UI & XML** - For a clean and intuitive user interface
- **MVVM Architecture** - To ensure maintainable and scalable codebase

---

## 🌟 **Features**
- **Track Movies:** Organize your shows into different categories like "Watched," "Currently Watching," "Plan to Watch," and "Favorites."
- **Movie Search:** Search for your favorite movies directly from TMDB.
- **User Authentication:** Firebase Authentication to securely sign in users and store personalized data.
- **Real-time Data Management:** Uses Firebase Realtime Database to keep your data synchronized across devices.
- **Simple & Clean UI:** Material UI and XML designs for an engaging and easy-to-use interface.

---

## 🌍 **API Used**
**TMDB API**  
The Movie Database (TMDB) is used to fetch real-time data of movies, allowing users to search and get details of their favorite movies.  
[Link to TMDB](https://www.themoviedb.org/)

---

## 🚀 **Motivation**
The motivation behind ShowTracker was to create a simple, yet efficient app for movie lovers who enjoy organizing and keeping track of their movie-watching habits. With future plans to add support for TV shows, this app will cater to all your entertainment tracking needs.

---

## 📋 **Future Plans**

## 🧰 **Project Setup**
1. Clone the repo:
   ```bash
   git clone https://github.com/devgupta4526/showtracker.git
   ```
2. Open the project in Android Studio.
3. Configure Firebase for Realtime Database and Authentication.
4. Set up the TMDB API key in the project.
